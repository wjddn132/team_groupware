＃ 유기견 봉사 활동을 위한 신청 플랫폼
사람 = 신청자 ( 이름, 성별, 나이(미성년 불가), 전화번호, 주소 ) 
강아지 = 거기 있는 유기견 ( 이름, 견종, 성별, 추정나이, 지병, 입소일, 입소사유, 정기후원자 명단, 예방접종 )
보호소 = 강아지가 속해 있는 지역별 보호소 ( 주소, 이름, 책임자, 계좌, 수용중인 개체수, 봉사 가능 요일 )
후원 = 보호소 (돈 혹은 용품 후원 가능) 혹은 강아지 한테 1:1로 돈을 후원하는 시스템

- user page
1. 회원 가입 ( 아이디, 이름, 성별, 나이, 전화번호, 주소 )
2. 근처 보호소 조회 -> 보호소 리스트 출력 -> 사용자가 원하는 보호소를 번호로 입력받기 -> ( 보호소 이름, 주소, 책임자 )는 기본으로 출력 -> ★1. 수용중인 유기견 리스트 확인 2. 봉사 가능 요일 조회 3. 후원 계좌 확인 0. 이전 화면으로
3. 봉사 신청 -> 보호소 리스트 출력 -> 봉사 가능 날짜 입력받기 -> 아이디 입력받기 -> if 미성년이 아닐 경우 -> 신청 완료!
if 신청자가 미성년자일 경우 ♣ 부모의 인적사항(이름, 나이, 전화번호, 주소)을 입력받는 페이지로 이동 ) -> if 정보 기입시 봉사 신청 가능
										              -> if 정보 미기입시 봉사 신청 불가능
4. 후원 신청 -> ♣ 신청자 정보 입력 받기 ( 아이디 ) -> ★ 1. 보호소에게 후원 2. 유기견에게 후원 -> if 보호소에게 후원 -> ★ 보호소 리스트 출력 후, 번호로 입력 -> ★1. 계좌로 후원 -> ♣보호소의 계좌 출력 및 금액 입력 받기 -> 감사 멘트
										         			                       	          2. 용품으로 후원 -> ★ 후원 용품의 품목 리스트를 출력 후, 번호로 입력받은 후, ♣수량까지 입력받기 -> 주소 띄우고 감사 멘트
						                                     -> if 유기견에게 후원 -> ★보호소 리스트 출력 후, 번호로 입력 -> ★유기견 리스트 출력 후, 번호로 입력 -> ♣보호소의 계좌 출력 및 금액 입력 받기 -> 감사 멘트 ((유기견의 이름)의 후원자가 되주셔서 감사합니다)
5. 회원 정보 조회 ( 아이디로 정보 조회 후 아이디, 이름, 성별, 나이, 전화번호, 주소, 봉사 내역( 보호소 이름, 날짜, 후원 여부( 보호소 후원 or 1:1 후원 ) )
6. 회원 정보 수정 ( 아이디로 정보 조회 후 아이디, 이름, 성별, 나이, 전화번호, 주소까지 출력 후 -> 각 항목 한 번씩 출력하며 재입력 받아 수정하기 )
7. 회원 정보 삭제
0. 프로그램 닫기
